### 三、部署启动说明
使用pnpm包管理工具
1、pnpm安装指令：
```
npm i -g pnpm
```
2、安装依赖
```
pnpm install
```
3、启动
```
pnpm run dev
```
4、构建生产环境
```
pnpm run build
```
5、访问
```
平台/租户管理员：土拨鼠充电公司
账户：admin    密码：admin
```
